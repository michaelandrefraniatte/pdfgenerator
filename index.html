PDFGenerator

PDF From HTML

Generate PDF From any Website

The program is a simple tool for having PDF files of website pages on your computer or online from an URL. Load the page, check it and convert it. It's very simple to debug the page you want a PDF. You can create a nice Curriculum Vitae or even a book to commercialize, with HTML and CSS, in PDF format, instead of with hilarious export, and with uncommercial use. Let's see...

Considerations About the Program

When loading a local file, the images shall be placed under assets folder from the program is. The image source URL is corresponding to https://appassets/ + image name + extension. You can put images in subfolders, and so add subfolder to image URL. But you can import a local HTML file directly from assets folder with relative links as shown with the test example present in the folder, allowing to display image. To note, when you generate PDF, it open the print user interface, there are options like format, portrait or landscape, or like show header information or not.
